# REST API: Проєкт SkillSwap



SkillSwap - це платформа для обміну навичками між підлітками. Уявіть: ви круто граєте на гітарі, але хочете навчитися програмувати. А ваш однокласник - навпаки, знає Python, але мріє про музику. SkillSwap допомагає вам знайти один одного та обмінятися знаннями!



## Основні можливості:

- Каталог навичок - користувачі додають, що вміють і чому хочуть навчитися
- Система matching - автоматичний пошук людей зі спільними інтересами
- Обміни - домовленості про взаємне навчання
- Відгуки - рейтингова система для підтвердження якості навчання


## MVP - Мінімально життєздатний продукт:

1. Налаштування проєкту - структура папок, віртуальне середовище
2. Базовий API для навичок - CRUD операції (Create, Read, Update, Delete)
3. Валідація даних - перевірка коректності введення через Pydantic
4. Фільтрація та пошук - знаходження навичок за категоріями
5. Документація - автоматична генерація через Swagger UI
6. Тестування - перевірка роботи через браузер
